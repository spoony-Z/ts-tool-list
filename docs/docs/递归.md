## 返回所有匹配到的节点数据

- **api：** **fileUrlType**
  
- 参数：
  - `list: Array` 递归的数组
  - `childKey: String` 子集字段
  - `key: String` 要匹配的字段名
  - `val: String | Number` 要匹配字段所对应的值
  - `list: String` 文件URL
  
- 示例
```js
import { getRecursiveMatch } from 'ts-tool-list';
function Recursion() {

  getRecursiveMatch(url)
}
```
